# Vietnamese Recipes

A simple HTML project created to practice the HTML concepts learned from **The Odin Project's** introductory course. This project contains a small website featuring recipes for some popular Vietnamese dishes.

## Contents

The website includes step-by-step recipes for:

- **Bún Chả**
- **Bún Bò Huế**

## Installation

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your web browser, or serve the project using a local web server (localhost).

## Technologies Used

- HTML5
